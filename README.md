### ‚öôÔ∏è Commands
| Commands | Description |
|---------|-------------|
| ```/afkzone pos1``` | Set Position 1 to current location |
| ```/afkzone pos2``` | Set Position 2 to current location |
| ```/afkzone create <name>``` | Create a new AFKZone |
| ```/afkzone delete <name>``` | Delete an AFKZone |
| ```/afkzone list``` | List AFKZone |
| ```/afkzone ui``` | Open AFKZone UI |
| ```/az``` | Alias Commands |
